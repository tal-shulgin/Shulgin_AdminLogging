# Shulgin_AdminLogging #

<b>this module will help admin to track all backend user activities.</b>

## tracking fetchers: ##
1. cms page edit
2. cms block edit
3. system config
4. product edit
5. category edit
6. custom modules


## future fetchers: ##
1. restore history data 


## composer require ##
<code>composer require symfony/var-dumper</code>
